## 1. 矛盾分析法

> "捉住了主要矛盾，一切问题就迎刃而解了。" ——《矛盾论》

**本质**：所有复杂问题背后都有一个"主要矛盾"。找到它，集中力量解决它，其他问题连带松动。

**操作步骤**：
1. **列出所有矛盾**——写成"A vs B"对立形式
2. **判断主要矛盾**——解决它能带动其他矛盾缓解的那个
3. **分析主要矛盾的两面**——哪面是主导因素（主要方面）
4. **区分性质**——敌我矛盾（斗争解决）vs 人民内部矛盾（说服教育解决）

**适用**：问题太多不知从何下手、需要聚焦决策
**AI时代补丁**：复杂系统可能没有单一"主要矛盾"——如果找不到，换系统思维

## 2. 实践认识循环

> "你要知道梨子的滋味，你就得变革梨子，亲口吃一吃。" ——《实践论》

**本质**：认识从实践来，经理论提升，回实践验证。螺旋上升。

**操作步骤**：
1. **调查研究**——没有调查就没有发言权。去一线，看数据，不要在办公室猜
2. **形成假设**——基于调查形成可验证的判断
3. **试点验证**——最小范围试，不求完美但求可测量
4. **总结提炼**——成功为什么成功？失败为什么失败？提炼规律
5. **再实践**——带着新认识进入下一轮

**适用**：方案争执不下、不确定方向对不对、需要学习新领域
**现代等价**：精益创业的build-measure-learn循环

## 3. 持久战略

> "最后胜利是中国的——这不是因为有更多的枪炮，而是因为我们有更多的人民。" ——《论持久战》

**本质**：敌强我弱时不求速胜，以时间换空间，在持久中改变力量对比。

**三阶段**：

| 阶段 | 力量对比 | 核心策略 | 禁忌 |
|------|---------|---------|------|
| 战略防御 | 对方明显强 | 保存实力、避决战、积蓄力量 | 禁硬拼、禁贪大 |
| 战略相持 | 差距缩小 | 扩根据地、消耗对方、等转折 | 禁急躁、禁松懈 |
| 战略反攻 | 我方占优 | 集中力量、决战决胜 | 禁骄傲、禁冒进 |

**操作**：判断当前处于哪个阶段→匹配该阶段策略→积蓄力量等转折点
**AI时代补丁**：winner-take-all市场中时间可能不站在弱者一边——叠加OODA/Blitzscaling判断

## 4. 农村包围城市

> "星星之火，可以燎原。" ——《星星之火，可以燎原》

**本质**：不攻最强点，先在边缘建根据地，积累到可以包围中心。

**操作步骤**：
1. **找"农村"**——对手忽视或不屑的市场/场景/用户群
2. **建根据地**——在这个边缘做到绝对第一
3. **根据地连片**——从一个边缘扩展到相邻的边缘
4. **包围城市**——积累足够后向核心市场进军

**适用**：市场进入、从0到1、面对巨头时的生存策略
**AI时代补丁**：网络效应可能让"农村"永远无法挑战有网络效应的"城市"——需检查

## 5. 统一战线

> "谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题。" ——《中国社会各阶级的分析》

**本质**：把朋友搞得多多的，敌人搞得少少的。

**操作步骤**：
1. **敌我友分析**——把利益相关方分为五档：坚定盟友、可争取的中间派、摇摆者、被动反对者、主要敌人
2. **找最大公约数**——在什么议题上可以团结最多人？
3. **又团结又斗争**——对盟友不无原则迁就，对可争取对象不放弃争取
4. **孤立主要敌人**——团结所有非主要敌人，使主要敌人陷入孤立

**适用**：联盟、谈判、职场人际、合作伙伴关系管理
**个人场景**：画出你职场中的"阶级分析图"——谁帮你、谁挡你、谁可以争取

## 6. 群众路线

> "从群众中来，到群众中去。" ——《关于领导方法的若干问题》

**本质**：最好的认知来自一线。收集分散意见→系统化提炼→带回一线验证。

**操作步骤**：
1. **深入一线**——去跟用户/员工/客户直接对话
2. **收集分散意见**——每个人都有碎片化的观察
3. **系统化提炼**——把分散意见整理成结构化认知
4. **带回一线验证**——让群众检验你的提炼是否准确

**适用**：产品决策、需求调研、团队管理
**现代等价**：用户研究 + 数据驱动决策 + PMF验证

## 毛泽东体系的适用边界

| 擅长 | 不擅长 |
|------|--------|
| 从0到1的竞争策略（打江山） | 从1到N的规模化治理（坐江山）→ 用邓小平 |
| 弱势方的非对称竞争 | 强势方的防御 → 用格鲁夫 |
| 战争/对抗性场景 | 合作/共赢/正和博弈 → 用现代平台理论 |
| 找方向、定策略 | 精细化管理、组织效能 → 用格鲁夫+德鲁克 |
| 直觉性的战略判断 | 理性化的决策质量控制 → 用芒格 |
