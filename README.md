# Strategic Thinking OS 🧠

**战略思维操作系统** — 五大思想体系 × 元路由器，为商业竞争、组织管理、个人成长、AI时代决策提供多框架分析。

## 五大思想体系

| 体系 | 核心问题 | 适用场景 |
|------|---------|---------|
| **毛泽东** | 弱者如何胜强？ | 竞争策略、市场进入、个人突围 |
| **邓小平** | 赢了之后怎么治理？ | 组织改革、制度优化、从1到N |
| **格鲁夫+德鲁克** | 如何管理并识别10x变化？ | 团队管理、OKR、AI转型 |
| **芒格** | 如何避免做蠢事？ | 重大决策、投资、认知偏差检查 |
| **现代方法论** | AI/互联网时代有何不同？ | OODA、精益、网络效应、AI竞争 |

## 核心特性

- **元路由器**：根据问题类型自动选择最佳框架组合，避免"锤子-钉子"效应
- **多框架交叉分析**：复杂问题用多个视角交叉检验，降低盲区概率
- **芒格检验层**：所有分析默认执行逆向思维、激励偏差、能力圈检查
- **经典+现代叠加**：经典框架打"AI时代补丁"，不教条不过时
- **完整案例**：Anthropic vs OpenAI 五体系联合分析示范


```bash

```

### 从 GitHub 安装

```bash
git clone https://github.com/tocekuma/strategic-thinking-os.git
cp -r strategic-thinking-os ~/.cursor/skills/
cp -r strategic-thinking-os ~/.claude/skills/
```

## 与 maoxuan-skill 的关系

- [maoxuan-skill](https://github.com/tocekuma/maoxuan-skill) 是毛泽东思想的深度单体系 skill（12个框架 + 原文论证链）
- strategic-thinking-os 是多体系 skill，包含精选的毛泽东框架 + 4个其他体系
- 如果你只想深入毛泽东思想 → 用 maoxuan-skill
- 如果你想多视角分析 → 用 strategic-thinking-os
